# Task 3 - Model Validation, Overfitting Control & Hyperparameter Tuning

## Objective
Enhance the California Housing Price Prediction system using Cross Validation, Overfitting Analysis and Hyperparameter Tuning.

## Dataset
California Housing Dataset

## Models Used
- Linear Regression
- Ridge Regression
- Decision Tree Regressor

## Techniques Implemented
- Train-Test Split
- Cross Validation
- Overfitting Analysis
- GridSearchCV
- Hyperparameter Tuning

## Results
- Best CV Score: 0.7105
- Tuned RMSE: 0.6389
- Tuned R² Score: 0.6885

## Best Hyperparameters
- max_depth = 10
